# Schema generation using Julia
- install Julia : [https://julialang.org/downloads/](https://julialang.org/downloads/)
- Open cmd
- run :`julia` and
  - `using Pkg`
  - `Pkg.add("JuMP")`
  - `Pkg.add("HiGHS")`
  - `Pkg.add("Dates")`
  - `Pkg.add("Printf")`
  - `Pkg.add("Plots")`
  - `Pkg.add("Measures")`
- install VSCode
- install Julia extension in VSCode
- go to `main.jl` in VSCode
